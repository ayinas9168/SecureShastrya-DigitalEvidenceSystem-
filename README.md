# SecureShastrya: Digital Evidence Locker & Management System

**SecureShastrya** is an advanced Android-based security platform designed to protect the "Chain of Custody" for digital evidence. Built for victims of cybercrime and legal investigators, it ensures that every photo, audio recording, and document captured is immutable, encrypted, and verifiable in a court of law.

## 🛡️ Key Features

*   **Secure In-App Capture:** Captures photos and audio directly within a secure buffer to prevent external tampering.
*   **Blockchain-Powered Integrity:** Records a unique SHA-256 hash of each file in a tamper-proof blockchain ledger.
*   **Military-Grade Security:** Double encryption using AES-256 and Android Keystore protection.
*   **Multi-Modal Authentication:** Supports standard password entry and high-security Biometric Login (Fingerprint/Face).
*   **Immutable Case Filing:** Synced to Cloud Firestore and Firebase Storage, creating permanent legal records.
*   **Automated Geotagging:** Every piece of evidence is automatically stamped with GPS coordinates and precise timestamps.

## 🚀 The MVP
SecureShastrya ensures digital evidence remains tamper-proof, providing a robust foundation for legal proceedings and shielding victims from data corruption.
